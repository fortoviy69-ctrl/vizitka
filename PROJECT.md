# Паспорт проекта: Цифровой двойник ведущего

> Последнее обновление: май 2026

---

## Название

**Цифровой двойник ведущего**

Рабочее название. Суть: ИИ берёт на себя ту часть работы ведущего, которую тот не успевает сделать в реальном времени — глубокий анализ каждого игрока.

---

## Что это

Telegram-бот с ИИ для ведущих трансформационной игры «Территория денег». Ведущий фотографирует заполненный бланк игрока — бот читает рукопись, анализирует паттерны, интерпретирует метафорические карты и выдаёт готовый персонализированный план действий на 30 дней. Ведущий получает структурированный результат за 1–2 минуты вместо 30–60 минут ручной работы.

---

## Для кого

**Основные пользователи — ведущие игры (платят и работают с ботом):**
- Профессиональные фасилитаторы — проводят групповые и индивидуальные сессии
- Коучи и психологи, использующие «Территорию денег» как инструмент работы
- Авторы и тренеры, обучающие новых ведущих

**Вторичные пользователи — игроки (получают результат, с ботом не работают):**
- Участники, которые хотят продолжить работу с инсайтами между сессиями

**Размер рынка (оценка):** несколько сотен сертифицированных ведущих «Территории денег» в РФ и СНГ. Рынок небольшой, но закрытый и лояльный — работают через сообщество правообладателя.

---

## Какую проблему решает

Трансформационная игра создаёт мощный эмоциональный опыт и инсайты — но они часто остаются «внутри» и не конвертируются в реальные изменения финансового поведения.

**Ведущий не может** в режиме реального времени делать глубокую интерпретацию для каждого из 5–8 игроков и одновременно вести группу. После сессии — устал, времени нет.

**Разрыв:** между «я понял что-то важное» и «я знаю, что конкретно сделать завтра».

ИИ-ассистент закрывает этот разрыв: берёт данные из бланка и карт, обрабатывает их в контексте финансовой цели игрока и выдаёт структурированный, практичный план — пока ведущий ещё на сессии.

## Конкуренты

Прямого конкурента нет. Ниша свободна.

| Продукт | Что делает | ИИ | Карты и игры | Рынок |
|---|---|---|---|---|

| ИгроN | Конструктор онлайн-игр | Нет | Частично | РФ/СНГ |
| Hedy AI | ИИ-ассистент для коуч-сессий (голос) | Да, сильный | Нет | Западный |
| CoachVantage | CRM для коучей | Слабый | Нет | Западный |
| Delenta | CRM для коучей с ИИ-резюме | Базовый | Нет | Западный |

**Главный вывод:** русскоязычные платформы работают с картами, но без ИИ. Западные ИИ-инструменты умеют анализировать сессии, но не знают трансформационных игр и русского контекста.

## Технологии

*Полный сравнительный анализ — в [TECH.md](TECH.md).*

### Рекомендованный стек MVP

| Компонент | Выбор | Цена/мес |
|---|---|---|
| OCR + Vision | Claude Vision (claude-sonnet) | ~$1–3 |
| LLM для анализа | Claude API (claude-sonnet) | ~$1–3 |
| База данных | SQLite → Supabase (миграция за 1 день) | $0 |
| Хостинг | Fly.io (бесплатный тир, без засыпания) | $0 |
| Telegram-бот | aiogram 3.x + FSM | $0 |
| **Итого** | | **~$2–6/мес** |

**Архитектура:**
```
Telegram (aiogram) → Claude Vision (OCR бланка) → Claude API (анализ + план) → SQLite → ответ ведущему
```

**Технический долг, который закладывается сразу:**
- SQLAlchemy как ORM — миграция SQLite → PostgreSQL без переписывания
- Конфигурация через `.env` — API-ключи не в коде
- Отдельный модуль LLM-провайдера — можно сменить без переписывания логики

**Главный практический риск:** оплата Anthropic API из РФ — нужна виртуальная карта Wise/Revolut или оплата через посредника.

## Монетизация

- Цена: ~7 900 ₽ разово (до 20 сессий/мес навсегда)
- Цель: 30–50 «основателей», которые тестируют и дают обратную связь
- Выручка: 237 000–395 000 ₽ стартового капитала

**Ключевой партнёрский канал:** официальный статус «Рекомендованный инструмент для сертифицированных ведущих» + revenue share 15–20% с подписки + включение в стоимость обучающего курса.

# Справочник: Механика игры

*Справочный раздел — нужен для написания промптов и контекста разработчика.*

## Структура бланка (8 страниц)

### Страница 1 — Старт
| Поле | Тип | Описание |
|---|---|---|
| Запрос | Текст | Финансовая цель или вопрос, с которым игрок пришёл на игру |
| Уровень старта | Один из четырёх | интуиция / действия / разум / энергия |
| Амулеты и денежные магниты | 5 карточек-изображений + подписи | Игрок отмечает полученные амулеты по ходу игры |

### Страницы 1–3 — Ходы
Таблица. Каждая строка = один ход:
| Колонка | Содержимое |
|---|---|
| Название клетки | Клетка, на которую попал игрок |
| 🪙 Проявленные деньги | Сколько реальных денег получил/потерял на ходу |
| 💎 Непроявленные деньги | Сколько потенциала/возможностей получил/потерял |
| ИНСАЙТ | Свободный текст — что осознал, что почувствовал |

### Страница 4 — Итоги игры
- Основные выводы — свободный текст, общее резюме всего опыта

### Страница 5 — План действий
- 5 действий для реализации запроса — 5 пронумерованных блоков по 3–4 строки

### Страница 6 — Декларация
- Имя, личная заметка, подпись, дата

## Механика двух валют (критично для интерпретации)

| Валюта | Символ | Значение в жизни |
|---|---|---|
| Проявленные деньги | 🪙 золотая монета | Реальный текущий доход |
| Непроявленные деньги | 💎 кристалл | Нереализованный потенциал: навыки, идеи, связи |

**Ключевая логика:** много кристаллов и мало монет = огромный потенциал, который игрок не монетизирует → прямой сигнал для плана действий.

---

## 4 уровня процветания (= 4 амулета = цель игры)

| Уровень | Суть | Амулет |
|---|---|---|
| Интуиция | Слышать себя, чувствовать куда двигаться | 🔵 |
| Действия | Эффективные действия, приводящие к доходу | 🟠 |
| Разум | Убеждения, мышление изобилия vs бедности | 🟣 |
| Энергия | Внутренний ресурс, забота о себе | 🔴 |

Нехватка амулета = слабое звено, на которое ИИ делает акцент в плане.

## Что ИИ получает на вход

1. Фото бланка (все страницы) → OCR → структурированный JSON
2. Фото метафорических карт → распознавание + интерпретация в контексте запроса
3. Итог: запрос + уровень старта + стартовый капитал + ходы (клетка + монеты/кристаллы + инсайт) + амулеты + выводы + черновые 5 действий

## Что ИИ выдаёт на выход

- **Анализ паттернов**: какие клетки повторялись, что это говорит о стратегии игрока
- **Ключевой блок**: какого амулета не хватало и почему
- **Соотношение валют**: вывод о непроявленном потенциале
- **Расширенный план**: черновые 5 действий → конкретный 30-дневный трек
- **Резюме для ведущего**: на чём сделать акцент в follow-up с игроком
